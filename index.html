AST

  • Abstract Syntax Tree.
  • Representación intermedia de alto nivel.
  • Retiene la estructura del lenguaje fuente.
  • Fácil de generar a partir del proceso de parseo.

AST

  • El objetivo es ocultar todo el overhead introducido por el parseo.
  • Obtener una representación del input, manteniendo la estructura de árbol, pero de forma simplificada.
  • Abstraer sólo la información relevante para el resto del análisis.
  • Ocultar todo aquello que ya no es interesante.

AST

  • Las gramáticas imponen estructura, es útil conservarla.
  • La sintaxis abstracta nos permite conservarla, dejando atrás la forma 'fea' en que se obtuvo.
  • El AST nos sirve para representar, y no para parsear.
